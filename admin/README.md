# VELORA Admin Panel

This is the administrative suite for the VELORA fashion e-commerce application.

## Tech Stack
- **Next.js 14** (App Router)
- **Tailwind CSS**
- **Firebase** (Auth, Firestore, Storage)
- **Lucide React** (Icons)

## Setup
1. Navigate to the `admin` directory.
2. Run `npm install` to install dependencies.
3. Update `src/lib/firebase.ts` with your actual Firebase project credentials.
4. Run `npm run dev` to start the development server.

## Features Implemented (Part 1)
- **Premium Login Interface**: Grayscale fashion editorial theme with secure admin-only access logic.
- **KPI Dashboard**: Real-time snapshot of products, orders, users, and revenue.
- **Recent Transactions**: Auto-updating table linked to Firestore.
- **Sidebar Navigation**: Desktop-first professional layout.
